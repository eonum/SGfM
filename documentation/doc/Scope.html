    class Scope

    Represents a Scope - a Scope can be attributed to Information, Relationship, etc, objects. It describes kind of a visibility filter. This class is meant to be extended. Extending classes need to implement the visible? method, which decides about visibility dependent on viewer, viewee and the field “list”.

def list_elements_proper
        allowed_elems =  [:Self, :Unauthorized] + Login.account_types.collect {|elem| elem.to_sym}
        unless list.all? { |elem| allowed_elems.include? elem}
                errors[:list] = "List elements aren't correct."
        end
end
